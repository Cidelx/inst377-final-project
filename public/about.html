About

This is a simple web application that helps students view and share feedback about college courses. Users can read reviews submitted by others and submit their own

What this does

• Lets users search universities
• Displays real course reviews
• Allows users to submit new reviews that are displayed

Why We Built It

This project was created for INST-377 to demonstrate full-stack web development. It connects a front-end website, a Node.js Express backend, and a Supabase database

Technology Used

• HTML, CSS, JavaScript
• Node.js + Express
• Supabase Database
• Fetch API
